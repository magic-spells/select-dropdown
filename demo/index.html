@magic-spells/select-dropdown

Dropdowns done
right

A fully accessible custom select component with keyboard navigation, ARIA support, and CSS-variable theming. No Shadow DOM — style it directly.

Example 01

Simple Select

A straightforward dropdown with a preselected value. Click or use keyboard to navigate options.

Cherry Apple Banana Cherry Durian Elderberry
Selected: Cherry
Example 02

Grouped Options

Organize options with <select-label> headings and <select-divider> separators. Non-option elements are skipped during keyboard navigation.

Select a snack Fruit Apple Banana Mango Strawberry Nuts Almond Cashew Pistachio Walnut
Selected: None
Example 03

CSS Variable Theming

Every color is driven by custom properties. Override them on a wrapper class to create distinct themes with no JavaScript.

Select a color Amber Crimson Jade Iris Slate

Form Integration

Accessibility

Markup
